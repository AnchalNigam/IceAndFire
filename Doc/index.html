IceAndFire App Urls

Home

http://iceandfire.webdeveloperjourney.xyz/home

This url gives list of categories like houses,books,characters with pagination.It is also having filters like filter by name, filter by category and filter by order.
Book

http://iceandfire.webdeveloperjourney.xyz/view/books/:booknumber

This url gives details of book and booknumber is route parameter which user will choose in home page.Whatever book user chooses, frontend will render that book info in detail.
House

http://iceandfire.webdeveloperjourney.xyz/view/houses/:housenumber

This url gives details of house and housenumber is route parameter which user will choose in home page.Whatever house user chooses, frontend will render that house info in detail.
